# Codex Prompt — Scaffold Implementation

Using the documentation in `docs/`, implement the first MVP scaffold for Life & Care Passport.

Implement:

- PHP 8.3 application bootstrap
- environment config loading from `.env`
- database connection layer
- simple routing
- base layout
- CSRF protection for forms
- error handling suitable for local development
- responsive starter CSS
- health/check page

Do not implement payments, AI, transcription, native app features, print integration, or organisational features.

After implementation, update documentation if any technical decisions differ from the plan.
