# Codex Prompt — Questionnaire Flow

Implement the MVP questionnaire flow.

Required behaviour:

- user can choose Essential 12, Recommended 25, or Full 37
- questions are loaded from database records
- one question is shown at a time
- answers autosave or save on navigation
- user can skip any question
- user can go back and edit
- progress indicator is visible
- skipped questions are recorded without being displayed awkwardly later

Do not add voice transcription or AI follow-up questions.
