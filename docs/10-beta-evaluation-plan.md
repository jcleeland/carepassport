# Beta Evaluation Plan

## Core Evaluation Question

Can Life & Care Passport create a respectful, readable, useful bedside identity package that families are proud to give and staff actually use?

## Family Evaluation

Ask family testers:

- Could you complete it without help?
- How long did it take?
- Which questions felt easy?
- Which questions felt repetitive, hard or upsetting?
- Did the Essential / Recommended / Full choice make sense?
- Did you understand the visibility settings?
- Was anything too private for the wall or bedside booklet?
- Would you feel comfortable giving the PDFs to a facility?
- What would you change before using this for someone you love?

## Staff Evaluation

Ask aged care staff or clinicians:

- Did you read the poster?
- Could you understand the person quickly?
- Which parts were most useful?
- Which parts were ignored?
- Was the wording respectful and non-directive?
- Did anything sound like a care instruction or clinical plan?
- Would this help you start a better conversation?
- Would this help you provide more person-centred support?

## Product Metrics

Track during beta:

- completion rate
- average completion time
- skipped question frequency
- most common visibility changes
- PDF generation success/failure
- user-reported output satisfaction

## Success Signal

The MVP is worth further investment if testers say:

- the process is completable
- the outputs are respectful and useful
- the posters are readable at a glance
- the booklet is coherent
- staff would actually use the documents
